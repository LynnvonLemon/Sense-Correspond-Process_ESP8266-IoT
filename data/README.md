# 配置文件说明

## WiFi设置
- `ssid`: WiFi网络名称
- `password`: WiFi密码

## 邮件设置
- `smtp.host`: SMTP服务器地址 (默认: smtp.163.com)
- `smtp.port`: SMTP端口 (SSL: 465, TLS: 587)
- `account.user`: 邮箱账号
- `account.password`: 邮箱授权码（不是登录密码）
- `sender.name`: 发件人显示名称
- `sender.email`: 发件人邮箱
- `recipients`: 收件人列表，可以添加多个
